the ``1-my_list`` module
======================

Using ``print_sorted``
-------------------

>>> MyList = __import__('1-my_list').MyList

empty list
>>> l = MyList()
>>> l.print_sorted()
[]

sort list when is unsorted
>>> l = MyList()
>>> l.append(5)
>>> l.append(2)
>>> l.append(7)
>>> l.append(6)
>>> l.append(8)
>>> l.print_sorted()
[2, 5, 6, 7, 8]

-int
>>> l = MyList()
>>> l.append(-5)
>>> l.append(-2)
>>> l.append(-7)
>>> l.append(-6)
>>> l.append(-8)
>>> l.print_sorted()
[-8, -7, -6, -5, -2]

only zero
>>> l = MyList()
>>> l.append(0)
>>> l.append(0)
>>> l.append(0)
>>> l.print_sorted()
[0, 0, 0]

only one int element
>>> l = MyList()
>>> l.append(2)
>>> l.print_sorted()
[2]

check if is subclcass
>>> issubclass(MyList, list)
True
